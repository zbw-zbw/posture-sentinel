# 体态哨兵 🦴

AI 实时坐姿检测工具，基于 Web 摄像头守护你的脊椎健康。

> 长时间久坐？低头驼背？体态哨兵通过摄像头实时分析你的坐姿，及时提醒，帮助你养成健康的坐姿习惯。

---

## 功能

- 📹 **实时坐姿检测** — 基于摄像头与 AI 姿态识别，实时分析头部角度、脊柱曲度等关键指标
- 🤖 **AI 姿态分析** — 利用 MediaPipe 关键点检测，精准识别身体姿态
- 📊 **健康日报** — 自动生成每日坐姿评分、分布统计与趋势分析
- 💡 **AI 健康建议** — 基于 DeepSeek 大模型，根据你的坐姿数据生成个性化的改善建议
- 🔒 **隐私优先** — 视频数据仅在本地处理，不录制、不上传；检测数据保存于浏览器本地存储
- ⚙️ **灵活设置** — 可自定义检测灵敏度、提醒阈值、通知方式等参数
- 📱 **响应式设计** — 完美适配桌面与移动端，随时随地关注脊椎健康

---

## 隐私

体态哨兵将用户隐私放在首位：

- **视频数据**：摄像头画面仅用于实时姿态分析，不录制、不截图、不上传到任何服务器
- **检测数据**：坐姿评分、指标数据仅保存在浏览器本地存储（localStorage）中，不传输到外部
- **AI 建议**：发送给 DeepSeek API 的仅是匿名化的数值统计数据（评分、角度、百分比），不包含任何图像或个人信息
- **数据清除**：你可以随时在设置中一键清除所有本地数据

## 技术栈

- **Next.js 16** — React 全栈框架，支持 App Router 与 API Routes
- **React 19** — 用户界面组件库
- **TypeScript** — 类型安全的开发体验
- **Tailwind CSS v4** — 原子化 CSS，快速构建现代化 UI
- **MediaPipe Tasks Vision** — Google 开源的计算机视觉模型，用于实时人体关键点检测
- **DeepSeek API** — 大语言模型，提供个性化健康建议

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

开发服务器启动后，在浏览器中访问 `http://localhost:3000` 即可预览。

---

## 部署

本项目推荐部署至 [Vercel](https://vercel.com)：

1. 将代码推送至 GitHub / GitLab / Bitbucket
2. 在 Vercel 中导入项目，框架自动识别为 Next.js
3. 在环境变量中添加：
   - `DEEPSEEK_API_KEY` — 你的 DeepSeek API 密钥，用于 AI 健康建议功能
4. 点击部署，即可上线

> 若无 DeepSeek API 密钥，AI 建议功能将不可用，但核心的实时检测与数据报表功能不受影响。
